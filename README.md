# 💼 AlkeWallet Business

Aplicación móvil orientada a la gestión de operaciones en punto de venta.

## 📱 Funcionalidades

- Registrar ingreso
- Pago a proveedor
- Visualización de saldo de caja
- Registro del último movimiento

## 🧠 Contexto

Proyecto basado en una experiencia real en tienda, considerando procesos como caja, proveedores y canales de venta digitales.
